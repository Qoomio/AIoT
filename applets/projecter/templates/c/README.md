# C Hello World Project

This is a simple C Hello World project created with the Projecter applet.

## Building the Project

```bash
make
```

## Running the Project

```bash
make run
```

Or directly:

```bash
./hello_world
```

## Cleaning Build Files

```bash
make clean
```

## Next Steps

- Edit `main.c` to add your code
- Modify the `Makefile` as needed for larger projects
- Consider adding header files in an `include/` directory
- Add unit tests with a testing framework like CUnit
